## Prerequisites

Before integrating Selecty with PageFly:

<Checklist>
  - [ ] [PageFly Landing Page Builder](https://apps.shopify.com/pagefly) installed
  - [ ] [Selecty](https://apps.shopify.com/selectors) installed and configured
  - [ ] At least one selector created in Selecty
  - [ ] Languages and/or currencies configured in Shopify Markets
</Checklist>

## Integration Setup

### Step 1: Create a PageFly Page

<Steps>
  <Step title="Open PageFly">
    In your Shopify admin, navigate to Apps → PageFly.
  </Step>

  <Step title="Create New Page">
    Click **Create your first page** or **New page**.
  </Step>

  <Step title="Select Page Type">
    Choose the type of page you want to create:
    - Regular page
    - Homepage
    - Product page
    - Collection page
    - Blog page
  </Step>

  <Step title="Choose Template or Start Blank">
    - Select from 100+ pre-designed templates, or
    - Start with a blank page for complete customization
  </Step>
</Steps>

### Step 2: Activate Selecty in PageFly

Before you can use Selecty selectors, you must activate the integration:

<Steps>
  <Step title="Access Third-Party Integrations">
    In PageFly's left sidebar:
    1. Scroll to **Third-party elements** section
    2. Click the **Third-party integrations** button
  </Step>

  <Step title="Find Selecty">
    In the integrations menu:
    1. Search for "Geolocation & Markets - Selecty"
    2. Locate the Selecty app in the list
  </Step>

  <Step title="Activate Integration">
    1. Click **Activate** on the Selecty integration
    2. Click **Done** to save
    3. Selecty elements now appear in the catalog
  </Step>
</Steps>

<Note>
You only need to activate Selecty once. After activation, Selecty elements will be available for all your PageFly pages.
</Note>

### Step 3: Add Selector to Your Page

<Steps>
  <Step title="Open Third-Party Elements">
    In PageFly's left sidebar:
    1. Click **Third-party integrations** button
    2. Search or scroll to find Selecty
  </Step>

  <Step title="View Available Selectors">
    Click on the Selecty integration to see:
    - All selectors you've created in Selecty app
    - Available selector variations
    - Preview of each selector
  </Step>

  <Step title="Add Selector to Page">
    1. Choose the selector you want to add
    2. **Drag and drop** it onto your page
    3. Position it where you want it to appear
  </Step>

  <Step title="Adjust Placement">
    - Drag to reposition
    - Resize if needed
    - Place in header, footer, or custom section
  </Step>
</Steps>

### Step 4: Create Selector in Selecty

If you haven't created selectors yet:

<Steps>
  <Step title="Open Selecty App">
    Go to Apps → Selecty in your Shopify admin.
  </Step>

  <Step title="Create New Selector">
    Click **Create selector** or **Create new**.
  </Step>

  <Step title="Configure Selector">
    1. Choose theme (Basic, Material, Cute, Minimal)
    2. Select resource type (Language, Currency, etc.)
    3. Pick selector type (Dropdown, Modal, etc.)
    4. Customize colors and appearance
  </Step>

  <Step title="Save and Publish">
    1. Click **Save**
    2. Selector becomes available in PageFly immediately
  </Step>
</Steps>

### Step 5: Customize Selector in PageFly

After adding a selector to your PageFly page:

<Steps>
  <Step title="Select the Selector Widget">
    Click on the selector element you added to your page.
  </Step>

  <Step title="Access Customization Options">
    In the right sidebar, you'll see:
    - **General tab** - Content, visibility, animation
    - **Styling tab** - Spacing, borders, opacity, CSS
  </Step>

  <Step title="General Settings">
    Configure:
    - Selector visibility rules
    - Animation effects
    - Display conditions
    - Mobile/desktop visibility
  </Step>

  <Step title="Styling Options">
    Customize:
    - Spacing (margins and padding)
    - Border styles
    - Background effects
    - CSS editor for advanced styling
  </Step>
</Steps>

<Tip>
You can customize selectors either in the Selecty app (affects all instances) or in PageFly (affects only that page). For page-specific styling, use PageFly's customization options.
</Tip>

### Step 6: Publish Your Page

<Steps>
  <Step title="Preview Your Page">
    - Click **Preview** to see how it looks
    - Test selector functionality
    - Check mobile and desktop views
  </Step>

  <Step title="Make Final Adjustments">
    - Fine-tune selector position
    - Adjust colors and spacing
    - Test language/currency switching
  </Step>

  <Step title="Publish">
    1. Click **Save** to save your changes
    2. Click **Publish** in the upper right corner
    3. Page goes live with selector
  </Step>
</Steps>

## Troubleshooting

<AccordionGroup>
  <Accordion title="Selecty not showing in PageFly integrations">
    **Solutions:**
    - Verify Selecty is installed in your Shopify store
    - Refresh PageFly page
    - Log out and back into PageFly
    - Contact PageFly support if issue persists
  </Accordion>

  <Accordion title="Selector not appearing on page">
    **Check:**
    - Selector is created and published in Selecty app
    - Integration is activated in PageFly
    - Widget is actually added to the page (check layers panel)
    - Visibility settings don't hide it
  </Accordion>

  <Accordion title="Selector not working correctly">
    **Verify:**
    - Languages/currencies configured in Shopify Markets
    - Selector settings are correct in Selecty app
    - No CSS conflicts with page styles
    - Test in different browsers
  </Accordion>

  <Accordion title="Styling conflicts">
    **Try:**
    - Adjust spacing in PageFly styling tab
    - Use Custom CSS to override conflicts
    - Simplify page layout around selector
    - Contact Selecty support for CSS help
  </Accordion>
</AccordionGroup>

## Advanced Customization

### Custom CSS in PageFly

For advanced users, add custom CSS to selectors:

1. Click selector in PageFly
2. Go to **Styling** tab
3. Find **CSS Editor** section
4. Add your custom styles

```css
/* Example: Custom selector styling in PageFly */
.selecty-selector {
  box-shadow: 0 2px 8px rgba(0,0,0,0.15);
  border-radius: 8px;
  padding: 4px;
}

.selecty-selector:hover {
  transform: scale(1.02);
  transition: all 0.2s ease;
}
```
