## Getting Support

### How to Contact Us

<Steps>
  <Step title="Choose Your Channel">
    Select the best support channel for your needs:
    - **Email** - For non-urgent, detailed questions
    - **Live Chat** - For quick questions and real-time help
    - **Phone** - For urgent or complex issues
    - **Documentation** - For self-service help
  </Step>

  <Step title="Gather Information">
    Before contacting support, collect:
    - Your store URL
    - Description of the issue
    - Screenshots or screen recordings
    - Steps to reproduce the problem
    - Error messages (if any)
    - What you've already tried
  </Step>

  <Step title="Describe Your Issue">
    Provide clear, detailed information:
    - What are you trying to do?
    - What's happening instead?
    - When did it start?
    - Does it happen every time?
    - Which browsers/devices are affected?
  </Step>

  <Step title="Get a Response">
    Our team will:
    - Acknowledge your request
    - Ask clarifying questions if needed
    - Provide solution or workaround
    - Follow up to ensure resolution
  </Step>
</Steps>

## What to Include in Support Requests

Help us help you faster by including these details:

### Essential Information

<Checklist>
  - [ ] Your Shopify store URL (example-store.myshopify.com)
  - [ ] Clear description of the issue or question
  - [ ] What you expected to happen vs. what actually happened
  - [ ] When the issue started
  - [ ] Steps to reproduce the problem
</Checklist>

### Helpful Additional Information

<Checklist>
  - [ ] Screenshots or screen recordings showing the issue
  - [ ] Browser and version (Chrome 120, Safari 17, etc.)
  - [ ] Device type (Desktop, iPhone 14, etc.)
  - [ ] Error messages or codes
  - [ ] Recent changes to your store or settings
  - [ ] What you've already tried to fix it
</Checklist>

## Common Issues and Quick Fixes

Before contacting support, try these quick fixes for common issues:

<AccordionGroup>
  <Accordion title="Selector not appearing on store">
    **Quick fixes:**
    1. Hard refresh your store (Ctrl+Shift+R or ⌘+Shift+R)
    2. Clear browser cache and cookies
    3. Test in incognito/private mode
    4. Verify selector is published in Selecty app
    5. Check that you have at least 2 languages or currencies configured
  </Accordion>

  <Accordion title="Wrong language or currency suggested">
    **For wrong language:**
    - Check your browser's language settings (not display language)
    - Verify language is configured in Shopify Markets
    - Test with different browser language preference

    **For wrong currency:**
    - Use VPN to test from different location
    - Verify country-to-market mapping in Shopify
    - Check IP geolocation at mylocation.org
  </Accordion>

  <Accordion title="Banner appearing every time">
    **Quick fixes:**
    1. Enable cookies in your browser
    2. Test in normal mode (not incognito)
    3. Check cookie settings in Selecty
    4. Clear existing cookies and test again
    5. Verify banner is set to appear "once per visitor"
  </Accordion>

  <Accordion title="Selector showing wrong options">
    **Quick fixes:**
    1. Sync Selecty with Shopify (refresh app)
    2. Verify resources in Shopify Markets settings
    3. Check selector resource configuration
    4. Ensure markets are published
    5. Verify currencies/languages assigned to markets
  </Accordion>

  <Accordion title="App loading slowly">
    **Quick fixes:**
    1. Clear browser cache
    2. Disable browser extensions temporarily
    3. Check internet connection
    4. Try different browser
    5. Contact support if issue persists
  </Accordion>
</AccordionGroup>

## Bug Reports

Found a bug? Help us fix it quickly:

### How to Report Bugs

<Steps>
  <Step title="Reproduce the Issue">
    Try to recreate the bug consistently:
    - Note exact steps to reproduce
    - Test in different browsers
    - Try incognito mode
    - Document when it happens
  </Step>

  <Step title="Gather Information">
    Collect these details:
    - Steps to reproduce
    - Expected behavior
    - Actual behavior
    - Screenshots or video
    - Browser and device info
    - Console errors (if technical)
  </Step>

  <Step title="Report the Bug">
    Contact support with:
    - Clear description
    - Reproduction steps
    - All gathered information
    - Impact on your store
    - Subject line: "Bug Report: [brief description]"
  </Step>

  <Step title="Follow Up">
    Our team will:
    - Confirm the bug
    - Provide workaround if available
    - Fix the issue
    - Update you when resolved
  </Step>
</Steps>

### Bug Priority

We prioritize bugs based on:

- **Critical** - Store is down or major functionality broken (fixed within hours)
- **High** - Significant feature not working (fixed within 1-2 days)
- **Medium** - Minor feature issue with workaround (fixed within 1 week)
- **Low** - Cosmetic or edge case issues (fixed in next update)
